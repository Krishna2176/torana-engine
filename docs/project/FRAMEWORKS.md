# TORANA Framework Overview

TORANA is composed of independent frameworks.

Each framework defines contracts.

Implementations build upon those contracts.

---

# Engine Core

Responsibilities

- Engine execution
- Scheduling
- Runtime
- Workflow execution

---

# Plugin Framework

Responsibilities

- Plugin contracts
- Plugin registration
- Workflow generation

---

# Dataset Framework

Responsibilities

- Dataset metadata
- Dataset discovery
- Dataset validation
- Provider contracts

---

# Analysis Framework

Responsibilities

- Analysis contracts
- Dataset requirements
- Analysis execution
- Workflow generation

---

# Provider Implementations

Responsibilities

- Dataset acquisition
- External system integration

Examples

- Local
- Google Earth Engine
- STAC
- PostGIS

---

# Visualization Framework

Responsibilities

- Maps
- Reports
- Three.js
- Blender
- Exports
