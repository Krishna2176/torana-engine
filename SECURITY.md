# Security Policy

## Reporting a Vulnerability

TORANA Engine is currently a private project under active development.

If you discover a security issue, please do **not** disclose it publicly.

Instead, contact the project owner directly with:

- description of the issue
- reproduction steps
- affected components
- suggested mitigation (if available)

Responsible disclosure is appreciated.

---

## Supported Versions

The project is currently under active development.

No public releases are supported at this time.
